# Project-on-Inventory-Management-System
Excited to share my Inventory Management System project, built as part of my Data Structures course at Daffodil International University!

This system efficiently manages products using a combination of Binary Search Trees, Hash Tables, and a Stack-based Undo feature to:

Add, delete, search, and update products in real-time
Maintain sorted inventory with fast lookup
Undo operations to ensure data safety

This project strengthened my understanding of core data structures and their practical applications in real-world systems.
